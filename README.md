## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/arka-garai/core-java.git
   ```
2. Open the project in your preferred IDE (IntelliJ IDEA recommended)
3. Ensure you have OpenJDK 21 installed
4. Navigate to the specific topic folder and run the Java files
