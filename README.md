# core-java

A comprehensive collection of Core Java concepts and examples for learning and reference.

## Requirements

- **Java Version**: OpenJDK 21

## Topics Covered

### 1. Object-Oriented Programming (OOPs)
Fundamental OOP concepts including classes, objects, inheritance, polymorphism, encapsulation, and abstraction.

**Folder**: `_00aObjectOrientedProgramming_OOPs`

### 2. Packages Demo
Understanding Java packages, package creation, and package management.

**Folder**: `_00bpackagesDemo`

### 3. Arrays
Working with arrays in Java, including single and multi-dimensional arrays.

**Folder**: `_00cArrays`

### 4. Abstract Classes and Interfaces
Implementation and usage of abstract classes and interfaces in Java.

**Folder**: `_00dAbstractClassAndInterface`

### 5. Enums and Exceptions
Exception handling mechanisms and enumeration types in Java.

**Folder**: `_00eEnumsAndExceptions`

### 6. Generics
Type-safe programming with Java Generics.

**Reference**: [Generics in Java](https://engineeringdigest.medium.com/generics-b158a743d18f)

### 7. Multithreading
Concurrent programming and thread management in Java.

**Reference**: [Multithreading in Java](https://engineeringdigest.medium.com/multithreading-in-java-39f34724bbf6)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/arka-garai/core-java.git
   ```

2. Open the project in your preferred IDE (IntelliJ IDEA recommended)

3. Ensure you have OpenJDK 21 installed

4. Navigate to the specific topic folder and run the Java files
